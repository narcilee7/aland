# ALand Design

**Agent Land（Aland）**——你的 AI Agent 数字大陆。

---

## 核心隐喻：一片被遗忘的数字大陆

你的电脑里有一片大陆，四个部落在此栖息。你不是管理员，你是**漫游者**。

| CLI | 部落 | 地形 | 生命体征 |
|-----|------|------|---------|
| Claude Code | **石塔城邦** | 沉稳的灰金色建筑群，中央有钟楼 | 进程运行时，钟楼指针转动；闲置时，钟摆缓慢摇晃 |
| Codex | **霓虹都市** | 赛博紫色网格，数据塔林立 | 运行时塔顶光束扫射；高负载时，整座城市"过载闪烁" |
| OpenCode | **开源荒野** | 青色帐篷与篝火，集市般松散 | 运行时篝火跳动；多人协作时，帐篷间亮起连接绳 |
| Kimi Code | **月白港湾** | 东方风格的港湾，帆船静泊 | 运行时白帆鼓动；闲置时，港湾起薄雾 |

大陆之间由**数据古道**连接。当两个 CLI 共享工作目录或配置时，道路亮起微光。

---

## 三大场景

### 1. 主界面：大陆俯瞰（The Overlook）

打开 Aland，不是弹窗，是**拉开一张地图**。

- **视角**：等距投影（Isometric），像 Mini Metro 或 Islanders 的极简风格，但更深邃
- **背景**：不是纯黑，是极深的靛蓝到墨绿渐变，像深夜从高空看大地
- **地形**：四个区域分布在大陆四角，有自然起伏——Claude 在高地，Codex 在平原网格，OpenCode 在河谷，Kimi 在海岸
- **天气系统**：
  - CPU 低负载 → 宁静，偶尔飘过数据云（半透明粒子）
  - CPU 高负载 → 局部区域风暴（该区域色调变深，粒子速度加快）
  - 深夜模式 → 大陆进入"月相"，各区域光源成为唯一照明

**交互**：鼠标滚轮缩放，拖拽平移。hover 区域时，地形轻微隆起，部落名称浮现（等宽字体，像地图标注）。

**底部状态栏**：一条细长的"罗盘"，显示当前系统时间、总 Agent 数、今日唤醒次数。

### 2. 降落：进入部落（The Landing）

点击区域，镜头从**俯瞰推近到地面**——不是页面切换，是空间穿越。

- Claude 石塔：推近时，钟楼放大，石砖纹理浮现，最终你站在塔底广场
- Codex 都市：推近时，网格线从平面变成立体街道，霓虹招牌逐个亮起
- OpenCode 荒野：推近时，篝火的火星溅到你眼前，帐篷帆布飘动
- Kimi 港湾：推近时，海浪声（可选）+ 白帆占据视野，码头木板清晰可见

**配置 = 地质层（The Strata）**

进入部落后，地面中央裂开，升起**三层旋转石盘/全息环**：

- **表层**：运行时参数（模型、温度、工作目录）——像风铃，可拨动
- **中层**：API 与权限——像锁芯，旋转解锁后显示
- **深层**：历史与记忆——像地层化石，层层叠叠

修改配置不是填表单，是**拨动、旋转、挖掘**。每个操作有物理反馈（Framer Motion spring）。

### 3. 唤醒与熄灭：生命仪式

**启动 CLI**：
- 回到俯瞰图，目标区域中心升起**图腾/烽火**
- Claude：钟楼钟声响起，指针飞速转动后稳定
- Codex：数据塔光束直冲天际，城市全面点亮
- OpenCode：篝火轰然炸开火星，帐篷群亮起暖光
- Kimi：港湾灯塔旋转，帆船白帆全部鼓起
- 一条光路从图腾延伸到底部"脐带"状态栏，显示 PID 与连接时长

**Kill 进程**：
- 长按区域，天气变暗，图腾颤抖
- 确认后，光束/火焰/钟声**碎裂成粒子**消散
- 区域回归宁静，但地面留下一道淡淡的**灼痕**（几分钟后消退）

**会话 = 记忆碎片（The Shards）**

散落在各区域中的发光碎片，像风滚草或萤火虫：

- 点击碎片，它展开成一卷**羊皮纸/全息屏**，显示历史对话摘要
- 碎片颜色深浅代表会话时长
- 搜索会话 = 释放一束探照光，被照亮的碎片高亮

## 技术骨架（Wails + React）

### 视觉层
- **地图引擎**：Canvas 2D 等距投影，不需要 Three.js（太重）。150 个地形粒子 + 四个区域精灵图足够
- **镜头推近**：Framer Motion `layoutId` 或自定义 spring 动画，从地图坐标系平滑过渡到区域坐标系
- **天气**：CSS 变量驱动，Go 后端每 2 秒推送 CPU/内存数据，前端映射到粒子速度和色调
- **字体**：地图标注用等宽（JetBrains Mono），部落内叙事文本用衬线（思源宋体或类似）增加文学感

### Wails 后端（Go）
```go
// 大陆心跳：每 2s 扫描进程，计算各部落"生命指数"
type TribeVital struct {
    Name      string  // claude | codex | opencode | kimi
    Running   bool
    CPU       float64 // 0-100，驱动天气强度
    Memory    float64
    CWD       string
    Uptime    int64
    // 生命指数映射到前端：呼吸频率、光束强度、篝火高度
}

// 配置地层：读取各 CLI 配置，解析为三层结构
type ConfigStrata struct {
    Surface map[string]interface{} // 运行时
    Middle  map[string]interface{} // API/权限
    Deep    []SessionMeta          // 历史
}
```

### 窗口体验
- **无边框**：`frameless: true`，自定义地图边框（像古老羊皮纸的烧焦边缘）
- **常驻**：系统托盘图标是**微缩大陆**，右键快速唤醒某个部落
- **全局快捷键**：`Cmd+Shift+A` 呼出悬浮地图（像 Spotlight，但你是从云端俯瞰）

## 为什么这是 Aland，不是管理系统

| 管理系统 | Aland |
|---------|-------|
| 列表 | 大陆 |
| 状态灯 | 钟楼指针、篝火、帆 |
| 编辑配置 | 拨动风铃、旋转锁芯 |
| 启动进程 | 点燃烽火 |
| Kill 进程 | 熄灭图腾，留下灼痕 |
| 查看历史 | 捡拾记忆碎片 |

**关键转折**：把"操作"变成**地理探索**。你不是在配置软件，你是在**漫游一片有生命的数字大陆**，与四个部落共处。
