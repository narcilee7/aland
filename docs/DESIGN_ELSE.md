## 一、大陆版图扩展

新增部落，按**生态聚落**分组：

| 部落 | 生态 | 地形 | 视觉特征 |
|------|------|------|---------|
| **Claude Code** | 古典派 | 石塔城邦 | 琥珀金钟楼，指针转动 |
| **Codex** | 工业派 | 霓虹都市 | 赛博紫数据塔，光束扫射 |
| **Cursor** | 现代派 | 玻璃穹顶 | 银蓝透明建筑，光标如流星划过 |
| **Trae** | 东方派 | 竹林庭院 | 翠竹与石板路，灯笼明灭 |
| **Kimi Code** | 港湾派 | 月白港湾 | 帆船与灯塔，白帆鼓动 |
| **OpenCode** | 开源派 | 荒野集市 | 篝火与帐篷，连接绳网 |
| **Windsurf** | 流动派 | 风之峡谷 | 流线型风帆建筑，气流可视 |
| **Aider** | 协作派 | 双生桥 | 两座塔由光桥连接，代表 pair |
| **Continue** | 嵌入式 | 地下根系 | 地表只有入口，内部是发光根系网络 |

**大陆布局**：不再四角，而是**环形大陆**——中央是圣所，各部落沿环分布，彼此有古道相连。

## 二、中央圣所：Token 熔炉（The Forge）

大陆正中央，一座悬浮的**水晶熔炉**——这是成本与 Token 的计算中心。

### 视觉
- 熔炉由半透明多面晶体构成，内部有**液态光**流动
- 颜色代表成本状态：
  - **翠绿**：今日花费健康
  - **琥珀**：接近预算阈值
  - **深红**：超出预算
- 熔炉表面有数字浮现（像全息投影），显示今日总 Token 消耗、预估月费

### 功能
- **实时流**：每个部落的 API 调用像**小溪汇入熔炉**，你可以看到哪条溪流最粗（哪个 CLI 最费 Token）
- **预算结界**：设置月度预算，接近时熔炉边缘出现**裂纹警告**
- **成本地层**：点击熔炉，地面升起环形地层，展示：
  - 按 CLI 分层的 Token 消耗（像地质剖面）
  - 按项目/工作目录的消耗热力图
  - 模型单价对比（GPT-4 vs Claude 3.5 vs 本地模型）

### 数据来源
- 读取各 CLI 的日志/历史记录解析 Token（Claude Code 有 `~/.claude/` 日志，Codex 有输出，Cursor 需要估算）
- 对于不暴露 Token 的（如 Cursor），基于字符数 + 模型价格表估算

## 三、灵动岛：天空之眼（The Eye）

Aland 常驻后台，以**系统级灵动岛**形式存在。

### 形态
- **macOS**：类似 Dynamic Island 的悬浮胶囊，位于菜单栏下方或屏幕边缘
- **Windows**：极简悬浮球，hover 展开

### 状态
| 状态 | 视觉 |
|------|------|
| 闲置 | 一颗微缩星球缓慢自转，代表 Aland 在沉睡守望 |
| 有 CLI 运行 | 星球表面出现**极光带**，颜色对应正在运行的部落 |
| 高 Token 消耗 | 星球表面出现**风暴漩涡** |
| 预算警告 | 星球闪烁红光，像脉冲星 |

### 交互
- **轻点**：展开迷你地图——显示当前运行的 CLI 缩略卡片，可直接 Kill 或切换
- **长按**：快速启动菜单——选择部落 + 工作目录，一键唤醒
- **拖拽**：把文件/代码片段拖到岛上，自动识别并询问"用哪个 CLI 处理？"

### 通知
- CLI 完成长时间任务 → 岛上绽放**烟花**（轻微动画）
- 成本突增 → 岛上出现**裂纹**并震动（视觉反馈）
- 配置冲突（两个 CLI 同时修改同一文件）→ 岛上出现**闪电**

## 四、交互深化：仪式系统

### 1. 唤醒仪式（Launch Ritual）
从 Eye 或地图启动 CLI：
1. 选择部落 → 地面升起对应图腾
2. 选择工作目录 → 一条光路从图腾延伸到目标位置
3. 确认 → 图腾**爆裂成光雨**，光雨落地后凝聚成该部落的"哨站"（代表进程）
4. 哨站与 Eye 之间出现**光桥**（实时数据流可视化）

### 2. 记忆碎片（Session Shards）
历史会话不再散落，而是**部落的集体记忆**：
- 每个部落有自己的**记忆树/记忆墙**
- Claude 石塔：钟声记录，每次会话是一个**钟摆轨迹**
- Cursor 穹顶：玻璃上凝结的**霜花图案**，每朵代表一次会话
- Trae 庭院：竹林中的**风铃**，每个风铃是一段对话

### 3. 配置 DNA（Config Helix）
进入配置不再裂地层，而是**召唤双螺旋**：
- 两条光带螺旋上升
- 一条代表当前配置，一条代表默认/推荐配置
- 差异点像**碱基对**发光，点击可切换
- 修改时，螺旋**重组**，有生物感

## 五、视觉系统规范

### 色彩
- **背景**：极深靛蓝 `#0a0e1a` → 墨绿 `#0d1f15` 渐变（深夜大陆）
- **部落色**：每个部落有主色 + 荧光 accent
  - Claude: `#d4a853` / `#ffdf80`
  - Cursor: `#60a5fa` / `#a5d8ff`
  - Trae: `#4ade80` / `#86efac`
  - Kimi: `#e2e8f0` / `#ffffff`
- **熔炉**：翠绿 `#22c55e` → 琥珀 `#f59e0b` → 深红 `#ef4444`

### 字体
- **地图标注**：JetBrains Mono，10px，微光效果
- **叙事文本**：思源宋体或 Playfair Display，增加文学感
- **数据**：SF Mono / Fira Code，等宽，数字变化用 spring 动画

### 动效原则
- **呼吸**：所有生命体 4s 周期
- **弹簧**：所有交互反馈用 `type: "spring", stiffness: 120, damping: 15`
- **粒子**：最多 200 个，Canvas 2D，拒绝 WebGL 过重
- **转场**：推镜（zoom in/out）代替页面切换，有**质量感**

### 音效（可选）
- 启动：低沉的钟声/引擎点火
- 配置修改：机械咔哒
- Kill：玻璃碎裂（轻微）
- 预算警告：心跳加速

## 七、MVP 路线

| 阶段 | 目标 | 可感知 |
|------|------|--------|
| **Week 1** | 大陆底图 + 4 个部落 | 打开 App，看到等距地图，四个区域可 hover |
